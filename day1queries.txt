create the index with shards and replicas

PUT /products
{
  "settings": {
    "number_of_shards": 1,
    "number_of_replicas": 1
  }
}

To check the index

http://localhost:9200/_cat/indices?v

To create the document with id

POST /products/_doc/1
{
  "productId":1001,
  "name":"laptop",
  "cost":45000
}
